1. Un ciudadano crea una causa en la plataforma. 
2. La causa se registra en la base de datos. 
3. Otros ciudadanos pueden apoyar la causa. 
4. El sistema cuenta la cantidad de apoyos recibidos.
5. Cuando se cumplen las condiciones, el sistema 
 genera un documento PDF con la petición para 
 presentarlo
