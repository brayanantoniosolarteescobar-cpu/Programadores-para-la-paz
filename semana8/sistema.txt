El comando uname -a muestra información del 
sistema operativo, como el nombre del sistema, 
versión del kernel, arquitectura y otros 
detalles técnicos. El comando whoami muestra el 
nombre del usuario actual que está utilizando el 
sistema es útil conocer esta información cuando se 
administra un servidor porque permite identificar el 
entorno de trabajo, verificar el sistema operativo y 
confirmar qué usuario tiene acceso al sistema.
